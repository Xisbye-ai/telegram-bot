# 🤖 Конструктор ботов

Программа для ПК, в которой ты **собираешь бота из блоков** — «найти картинку», «кликнуть»,
«если не нашёл — искать дальше» — и можешь **обучить нейросеть** находить объекты, которые
каждый раз выглядят по-разному (цвет, форма, положение). Интерфейс открывается в браузере,
поэтому **с телефона тоже видно**, как бот устроен и что он делает прямо сейчас.

Всё работает локально на твоём ПК, никуда в интернет ничего не отправляется.

## Установка (один раз)

1. Установи [Python 3.10+](https://www.python.org/downloads/) (на Windows при установке
   отметь галочку **Add Python to PATH**).
2. Открой командную строку в папке `bot-builder` и выполни:

   ```
   pip install -r requirements.txt
   ```

   Скачается примерно 400 МБ (больше всего весит PyTorch — он нужен для нейросети;
   если нейросеть не нужна, удали строку `torch` из `requirements.txt`).

## Запуск

```
python run.py
```

В консоли появятся два адреса:

- **На этом ПК** — открой `http://localhost:8000` в браузере;
- **С телефона** — открой показанный адрес вида `http://192.168.x.x:8000`
  (телефон должен быть в том же Wi-Fi; если не открывается — разреши Python
  доступ к сети в брандмауэре Windows, окно с вопросом появляется при первом запуске).

С телефона видно всё то же самое: блоки сценария, журнал работы бота в реальном времени,
живой просмотр экрана ПК (вкладка «Экран» → «Живой просмотр»). Можно даже управлять:
запускать/останавливать бота и запускать обучение.

> ⚠️ Пароля нет, поэтому запускай только в домашней сети, которой доверяешь.
> Если доступ с телефона не нужен: `python run.py --host 127.0.0.1`

## Как собрать бота (вкладка «Редактор»)

Бот выполняет блоки сверху вниз. Кнопка «＋ Добавить блок» есть в конце списка и внутри
блоков-контейнеров. Нажми «✨ Пример», чтобы увидеть готовый сценарий.

| Блок | Что делает |
|---|---|
| 🔍 Найти картинку | Ищет на экране сохранённый кусочек экрана (образец). Режим «Ждать, пока появится» — ищет снова и снова, пока не найдёт |
| 🧠 Найти объект (нейросеть) | То же, но ищет обученная нейросеть — для объектов, которые выглядят по-разному |
| 🖱 Клик | Кликает по тому, что нашлось, или по координатам |
| ⌨️ Ввести текст / Нажать клавишу | Печатает текст, жмёт клавиши (`enter`, `esc`, `ctrl+c`…) |
| ⏳ Пауза, 🖲 Прокрутка | Ждёт, крутит колесо мыши |
| 🔀 Если найдено… | Две ветки: что делать, если нашлось, и что — если нет |
| 🔁 Повторить N раз, ♾ Повторять бесконечно | Циклы |
| 💬 Сообщение, ⏹ Стоп | Пишет в журнал, останавливает бота |

Типичный сценарий «искать, пока не найдёшь, потом кликнуть»:

```
♾ Повторять бесконечно
   🔍 Найти картинку (режим: ждать, пока появится)
   🔀 Если найдено
      ✅ → 🖱 Клик по найденному → ⏳ Пауза 2 сек
```

Картинки-образцы делаются на вкладке **«Экран и образцы»**: сделай снимок, выдели
нужное место мышью, дай имя, сохрани. Кнопка «Через 5 сек» даёт время переключиться
на окно игры.

## Как обучить нейросеть (вкладка «Обучение»)

Для объектов, которые каждый раз разные (например, камушки в игре):

1. **Собери примеры.** Вкладка «Экран» → сделай снимок → режим «Разметка» → обведи
   каждый камушек рамкой (класс, например, `камушек`) → «📚 В обучение».
   Повтори на 5–10 разных снимках, чтобы набралось 30–50 рамок. Чем разнообразнее
   примеры (разный фон, цвет, размер), тем лучше нейросеть поймёт, что такое «камушек».
2. **Обучи.** Вкладка «Обучение» → имя модели → «🧠 Обучить». На обычном ПК это
   несколько минут; ход виден по шкале и в журнале.
3. **Проверь.** Кнопка «🔎 Найти на экране сейчас» покажет, что модель находит.
   Плохо находит — добавь ещё снимков с рамками и обучи заново. Находит лишнее —
   подними «уверенность».
4. **Используй.** В редакторе добавь блок «🧠 Найти объект (нейросеть)» и выбери модель.

В одну модель можно разметить несколько классов (камушки, монетки…) — в блоке поиска
потом выбирается, что именно искать.

## Полезно знать

- **Аварийный стоп:** резко дёрни мышь в левый верхний угол экрана — бот сразу остановится.
  Кнопка «⏹ Стоп» тоже работает (в том числе с телефона).
- Бот ищет и кликает на **основном мониторе**.
- Образец ищется «как есть», поэтому если поменять масштаб/разрешение игры — сохрани
  образец заново. Нейросеть к таким изменениям устойчивее.
- Нейропоиск заметно медленнее поиска по картинке (секунды на проверку) — это нормально.
- Все твои данные лежат в папке `bot-builder/data`: сценарии, образцы, снимки для
  обучения и модели. Можно копировать как резервную копию.

## Как это устроено (для любопытных)

- `run.py` — запуск локального веб-сервера (FastAPI + uvicorn).
- `app/main.py` — API: сценарии, снимки экрана, образцы, датасет, обучение, журнал (WebSocket).
- `app/engine.py` — исполнитель блоков: поток, который снимает экран (`mss`), ищет
  (`OpenCV` или нейросеть) и управляет мышью/клавиатурой (`pyautogui`).
- `app/detector.py` — нейросеть на PyTorch: маленькая свёрточная сеть учится отличать
  кусочки экрана 64×64 («фон или объект»), а потом за один проход строит карту
  вероятностей по всему экрану в нескольких масштабах.
- `app/static/` — весь интерфейс: HTML + CSS + JavaScript без сторонних библиотек.
