# 🦊 Английский с Фокси

Мультяшное приложение для изучения английского **с нуля** — один файл `index.html`,
работает на телефоне и компьютере, без интернета и без установки. Весь прогресс
сохраняется на устройстве автоматически.

## Что внутри

- **319 слов** в 15 темах: приветствия, семья, числа, цвета, еда, животные, дом,
  одежда, город, тело, природа, время, глаголы, прилагательные, школа.
  У каждого слова: перевод, транскрипция русскими буквами, картинка-эмодзи,
  живой пример с переводом и озвучка.
- **14 уроков грамматики** с нуля: местоимения, am/is/are, вопросы и отрицания,
  артикли, множественное число, this/that, my/your, Present Simple, don't/doesn't,
  Do/Does, there is/are, can, предлоги места, Present Continuous. Каждый урок —
  короткое наглядное объяснение и весёлая практика.
- **6 типов упражнений**: карточка нового слова, перевод в обе стороны,
  аудирование («что ты услышал?»), сборка слова по буквам, заполнение пропусков,
  сборка предложений.

## Научная база (почему это работает)

| Методика | Как реализована |
|---|---|
| Интервальные повторения (spaced repetition) | Каждое слово возвращается через 1 → 3 → 7 → 14 → 30 дней, точно перед тем как забудется. Кнопка «Повторить» показывает, что пора освежить |
| Активное вспоминание (active recall) | Не перечитывание, а тесты: мозг сам достаёт слово из памяти |
| Двойное кодирование (картинка + слово) | У каждого слова есть образ-эмодзи и озвучка |
| Контекст | Каждое слово даётся в живом предложении |
| Микро-уроки | Порция — 5 слов, урок занимает ~5 минут |
| Геймификация | Стрик 🔥, XP, уровни от Яйца 🥚 до Легенды 👑, 15 наград, цель дня |
| Мгновенная обратная связь | Ошибся — сразу видишь правильный ответ, и задание вернётся в конце урока |

## Как открыть

1. **Локально** — просто открой `english/index.html` в любом браузере.
2. **Через хостинг бота** — после деплоя приложение доступно по адресу
   `https://<твой-домен>/english`.
3. **Как кнопку в Telegram-боте** — в @BotFather: *Bot Settings → Menu Button →
   указать URL* `https://<твой-домен>/english`. Приложение откроется внутри
   Telegram как мини-приложение.

## Технически

- Один самодостаточный HTML-файл: без зависимостей, CDN и сборки.
- Прогресс — в `localStorage` браузера (устройство-локально).
- Озвучка — встроенный в браузер синтез речи (Web Speech API), офлайн.
- Светлая и тёмная тема — автоматически по настройке системы.
